# Zhioise Website V3

Static bilingual website for Zhioise / 青止. No external fonts, libraries or image hosting are required.

## Upload to GitHub / Vercel

1. Unzip the package.
2. Upload **the contents inside `zhioise-website-v3`** to the root of the GitHub repository.
3. Replace the previous HTML/CSS/JS files and upload the full `assets` folder.
4. Make sure `index.html` is visible at the repository root.
5. Vercel should redeploy automatically after the commit.

## Stable content model

- Product pages can be added gradually.
- Experience themes remain evergreen and are arranged by inquiry.
- Tea remains an inquiry page until product photos are ready.
- The founder page is intentionally linked only from About and the footer.

## Contact email

The current email is `hello@zhioise.com`. To change it, edit the first line of `assets/js/site.js` and replace occurrences in the HTML files.
