# Zhioise Website V1

Static website ready for GitHub Pages.

## Before publishing
1. Replace `hello@zhioise.com` in `index.html` with your actual Zoho business email if different.
2. Add Instagram / Xiaohongshu links if desired.
3. Review product descriptions and update product images whenever collections change.

## GitHub Pages
Upload all files to the root of a GitHub repository. In Settings → Pages, publish from the `main` branch and root folder.
