# Zhioise Website V2

This package includes all website pages and all image assets. Images use relative paths such as `assets/images/...`, so they will display correctly when the entire folder contents are uploaded to GitHub or Vercel.

## Upload to GitHub

1. Download and unzip `zhioise-website-v2.zip` on your computer.
2. Open the unzipped `zhioise-website-v2` folder.
3. In your GitHub repository, delete the old standalone `index.html` if you are replacing it.
4. Click **Add file → Upload files**.
5. Drag **all contents inside the unzipped folder** into GitHub: the HTML files, `styles.css`, `script.js`, and the entire `assets` folder.
6. Do not upload only the ZIP file.
7. Commit the upload.

## Why the V1 images did not appear

The ZIP was uploaded as one file. Hosting platforms do not unzip repository files automatically. They need the `assets/images` folder and the website files to exist separately in the repository.
