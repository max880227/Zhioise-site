青止 Zhioise 网站 V3 上传步骤

1. 下载并解压 zhioise-website-v3.zip。
2. 打开解压后的 zhioise-website-v3 文件夹。
3. 进入 GitHub 的 Zhioise-site 仓库。
4. 点击 Add file → Upload files。
5. 将文件夹里面的全部内容拖进去：
   - index.html
   - 其他 .html 页面
   - assets 文件夹
   - vercel.json
   - robots.txt
   - sitemap.xml
   不要只上传 zip，也不要让 index.html 多套一层文件夹。
6. 页面下方填写 Commit message，例如：Upload Zhioise V3。
7. 点击 Commit changes。
8. 等待 Vercel 自动重新部署，通常几分钟内完成。
